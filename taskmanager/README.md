# Task Manager API

Това е базов Spring Boot REST API проект за управление на задачи (Tasks).

## Как работи

- Можеш да създаваш нови задачи (POST /tasks)
- Можеш да виждаш всички задачи (GET /tasks)

## Технологии

- Java 17
- Spring Boot
- Spring Data JPA
- H2 in-memory Database
- Maven

## Стартиране

1. Отвори проекта в IntelliJ
2. Стартирай `TaskmanagerApplication.java`
3. В браузъра отиди на http://localhost:8080/tasks
